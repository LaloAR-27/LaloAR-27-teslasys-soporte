# TeslaSys Chile - Sistema de Registro de Solicitudes de Soporte

Aplicación web para registrar y visualizar solicitudes de soporte técnico de los colaboradores de TeslaSys Chile.

**Evaluación Sumativa 2** - Asignatura: Desarrollo Frontend - INACAP

## Tecnologías

- HTML5
- CSS3 (diseño responsivo, animaciones, variables CSS)
- JavaScript (ES6+) sin frameworks
- GitHub Copilot como apoyo al desarrollo

## Funcionalidades

- Formulario para ingresar solicitudes de soporte con 5 campos.
- Validación de campos obligatorios y formato de correo electrónico.
- Almacenamiento de solicitudes en un arreglo de objetos en memoria.
- Tabla dinámica que se actualiza automáticamente sin recargar la página.
- Eliminación individual de solicitudes registradas.
- Contador en tiempo real de solicitudes totales.
- Mensajes de éxito y error mostrados en pantalla mediante el DOM.
- Diseño responsivo adaptable a móviles, tablets y escritorio.

## Cómo ejecutar

1. Clonar este repositorio o descargar los archivos.
2. Abrir `index.html` con cualquier navegador moderno (Chrome, Firefox, Edge).
3. No requiere servidor ni dependencias externas.
